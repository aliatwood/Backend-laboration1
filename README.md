# Laboration 1 - Serverbaserad webbutveckling

## Beskrivning
I denna laboration skapas en webbplats där användaren kan lagra och ta bort kurser.
Jag har använt mig av Express och Node.js för att skapa en server kopplad till en SQLite-databas.
